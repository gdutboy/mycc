# 桌面操控技能 - 远程桌面调试任务

## 问题描述

在远程桌面 (RDP) 环境下，截图和 OCR 识别存在以下问题：

1. **截图捕获的是远程桌面窗口内容**，不是实际显示器
2. **坐标偏移**：点击坐标与实际桌面图标位置不一致
3. **分辨率差异**：远程桌面分辨率与实际屏幕可能不同

## 调试方向

1. **验证本地操作**：不在远程桌面时测试桌面操控是否正常
2. **坐标校准**：在远程桌面环境下建立坐标映射关系
3. **检测远程桌面**：添加检测逻辑，自动适配远程/本地环境
