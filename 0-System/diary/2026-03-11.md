## 今日变更

### Windows Toast 通知系统接入（paceaitian/claude-code-windows-toast-notification）

**功能**：多终端运行 CC 时，任意终端需要用户确认 → 弹出 Windows Toast 通知 → 点击跳转对应终端

**安装路径**：
- 脚本：`~/.claude/hooks/notification-system/`
- pwsh：`AppData\Local\Microsoft\PowerShell\7\pwsh.exe`
- BurntToast：1.1.0（CurrentUser）
- URI 协议：`claude-runner://` 已注册

**Hook 配置**（`~/.claude/settings.json`）：
- `Notification` Hook → `Launcher.ps1 -Delay 10`
- `Stop` Hook → `Launcher.ps1 -Delay 20`

### 禁用旧通知方案（mycc/notification.ts）

**原因**：昨天自研的 `notification.ts` 与 Hook 方案重叠，Stop 事件会触发两条通知

**改动**：
- `index.ts` 注释掉 `import { initNotificationSystem, sendNotification }`
- 移除启动通知、停止通知调用
- `sendStopNotification` 改为空函数，保留调用点不动
