# 读取公众号文章

给 cc 一个微信公众号链接，自动抓取文章内容（包括配图），生成结构化总结。

## 触发词

- `/read-gzh <链接>`
- `/read-gzh <链接1> <链接2> ...`（批量处理）
- "帮我读一下这篇公众号"
- "总结一下这篇文章"

## 执行流程

**Step 1: 抓取文章**

```bash
python3 .claude/skills/read-gzh/scripts/fetch_wechat_article.py "<公众号链接>"
```

获取：标题、作者、完整正文、所有配图链接。

**Step 2: 下载并识别配图**

创建临时目录，下载所有配图，读取每张图片提取文字/关键信息，纳入总结。

**Step 3: 生成结构化总结**

输出：基本信息 → 核心观点 → 关键信息 → 金句摘录 → 图片内容 → 思考/迭代点。

**批量处理**：逐篇处理后追加"汇总洞察"（共同主题、差异点）。

## 边界

- 执行时间：≤5 分钟/篇，批量每篇 +3 分钟
- 图片：≤20 张/篇
- 视频号只能获取简介；需要登录的内容无法获取
