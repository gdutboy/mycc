# Outline - 文章大纲生成

> 根据主题生成结构化的文章大纲。

## 触发词

- "大纲"
- "写大纲"
- "outline"
- "帮我列个提纲"

## 输入

用户提供：
- 文章主题
- 目标读者（可选）
- 文章类型（可选）：干货文、观点文、故事文等

## 输出

结构化大纲，格式：

```markdown
## [文章标题]

### 引言
- xxx

### 一级标题 1
#### 1.1 二级标题
- 论点1
- 论点2

#### 1.2 二级标题
- 论点1

### 一级标题 2
...

### 总结
- xxx
```

## 大纲原则

### 1. 逻辑清晰

- 层层递进
- 平行并列
- 过渡自然

### 2. 重点突出

- 核心观点前置
- 重要章节多展开

### 3. 读者导向

- 考虑读者认知水平
- 难易结合
