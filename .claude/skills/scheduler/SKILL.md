# 定时任务系统

定时任务功能**内置在 mycc 后端**，启动后端后自动生效。

---

## 执行流程

收到用户指令后按以下映射执行：

| 用户说 | AI 执行 |
|--------|----------|
| "启动定时任务" | 执行 `/mycc` 启动后端 |
| "停止定时任务" | 停止后端 |
| "查看定时任务" | 读取并显示 `tasks.md` |
| "添加定时任务 xxx" | 编辑 `tasks.md` 添加一行 |
| "每天 9 点提醒我 xxx" | 添加 `\| 09:00 \| xxx \| /tell-me \| ... \|` |

---

## 任务配置位置

- 任务列表：`.claude/skills/scheduler/tasks.md`
- 执行历史：`.claude/skills/scheduler/history.md`（自动创建）

---

## 时间格式速查

| 类型 | 示例 |
|------|------|
| 每日 | `08:00` |
| 每周 | `周一 09:00` |
| 一次性 | `2026-02-01 10:00` |
| 间隔 | `每30分钟`、`每2小时` |
