# 定时任务配置

> 使用方法：启动 mycc 后端后，定时任务自动生效

---

## 每日任务

| 时间 | 任务 | Skill | 说明 |
|------|------|-------|------|
| 06:00 | 每日早报 | /collect | 采集时政热点、AI动态、股票要闻，生成简报推送到飞书 |
| 08:00 | CC学习 | - | 读取 3-Thinking/claude-code学习指南.md，按进度生成当日学习内容到 0-System/cc-today-learn.md。要求：①不是简单复制原文 ②增加自己的解释、例子、练习建议 ③最后写一句今日思考 ④明日预告下周内容 |
| 22:00 | 根目录清理 | - | 检查项目根目录，按文件分类规则清理临时文件：① 移动分类不对的文件到正确目录（如 tasks/、scripts/）② 删除调试截图、测试输出等无用文件（png/txt/debug.html）③ 汇报清理结果 |

## 时间格式说明

| 类型 | 格式 | 示例 |
|------|------|------|
| 每日 | HH:MM | `08:00`、`22:30` |
| 每周 | 周X HH:MM | `周一 09:00`、`周日 20:00` |
| 一次性 | YYYY-MM-DD HH:MM | `2026-02-01 10:00` |
| 间隔 | 每X分钟 / 每X小时 | `每30分钟`、`每2小时` |

---

## 注意事项

1. **Skill 列**：填写要执行的 skill 名称（如 `/tell-me`），填 `-` 表示无特定 skill
2. **说明列**：会传给 AI 作为任务描述，可以写详细需求
3. **时间误差**：允许 ±2 分钟误差
4. **一次性任务**：执行后每日会自动清理，避免累计太长
