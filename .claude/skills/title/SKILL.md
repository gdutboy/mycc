# Title - 公众号标题生成

> 为公众号文章生成吸引人的标题。

## 触发词

- "起标题"
- "生成标题"
- "帮我取个标题"
- "title"

## 输入

用户提供：
- 文章主题/核心观点
- 目标读者
- 想要的风格（可选）

## 输出

生成 5-10 个标题选项，格式：

```markdown
## 标题候选

### 1. [标题1]
特点：xxx

### 2. [标题2]
特点：xxx

...
```

## 标题原则

1. **引发好奇**：让人想点进来
2. **明确价值**：让人知道能得到什么
3. **营造氛围**：情绪感强
4. **简洁有力**：不超过 25 字

## 示例

输入：分享 Claude Code 使用技巧的文章

输出：
1. "我是如何用 AI 一个人顶一个团队的"
2. "Claude Code 实战经验分享"
3. "从 0 到 1：我的 AI 协作方法论"
4. "一个人就是一个团队：我的 AI 提效实践"
5. "你真的会用 Claude Code 吗？"
