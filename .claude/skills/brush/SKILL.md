# Brush - AI 图片生成

> 根据描述生成配图。

## 触发词

- "生成图片"
- "画图"
- "配图"
- "brush"

## 输入

用户提供：
- 图片主题/内容描述
- 风格要求（可选）
- 尺寸（可选）

## 输出

生成的图片链接 + 提示词。

## 支持的风格

| 风格 | 描述 |
|------|------|
| 写实 | 照片级真实感 |
| 插画 | 扁平化/矢量风格 |
| 漫画 | 漫画/二次元 |
| 极简 | 简约线条 |
| 抽象 | 艺术抽象 |

## 使用场景

- 文章配图
- 社交媒体配图
- 封面图
- 演示插图

## 注意事项

- 描述越具体效果越好
- 包含主体、背景、风格、颜色等细节
