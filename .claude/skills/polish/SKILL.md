# Polish - 文章精修

> 打磨文章语言，提升可读性和说服力。

## 触发词

- "精修"
- "打磨"
- "润色"
- "polish"
- "帮我改改"

## 输入

用户提供需要精修的文章（Markdown 格式）。

## 输出

精修后的文章 + 修改说明。

### 修改说明格式

```markdown
## 修改说明

### 1. 结构调整
- 原文：xxx
- 修改：xxx
- 理由：xxx

### 2. 语言优化
- 原文：xxx
- 修改：xxx
- 理由：xxx

### 3. 亮点增强
- 新增：xxx
- 理由：xxx
```

## 精修维度

### 1. 语言流畅

- 去除冗余表达
- 优化句式结构
- 保持节奏感

### 2. 逻辑清晰

- 论证链条完整
- 过渡自然
- 重点突出

### 3. 亮点突出

- 强化核心观点
- 增加记忆点
- 提升说服力

### 4. 读者视角

- 术语解释
- 例子补充
- 降低理解门槛

## 注意事项

- 保持作者原意
- 不要过度华丽
- 实用为主
