# Create System - 创作系统路由

> 创作领域的统一入口，根据任务类型路由到具体技能。

## 触发词

- "写文章"
- "创作"
- "create-system"
- "写公众号"

## 路由规则

根据任务类型，选择对应技能：

| 任务类型 | 使用技能 |
|---------|---------|
| 起标题 | title |
| 写大纲 | outline |
| 写初稿 | draft |
| 精修文章 | polish |
| 排版渲染 | gzh |
| 读公众号文章 | read-gzh |
| 生成配图 | brush |

## 执行流程

### Step 1: 判断任务类型

理解用户要做什么，确定需要哪个技能。

### Step 2: 加载对应技能

调用对应的 Skill 获取指引。

### Step 3: 执行并交付

按技能指引执行，返回结果。

## 子技能说明

### title - 标题生成

根据文章主题生成多个标题选项。

### outline - 大纲生成

根据主题生成文章大纲结构。

### draft - 初稿生成

根据大纲写初稿。

### polish - 精修

打磨文章：语言流畅、逻辑清晰、亮点突出。

### gzh - 公众号排版

渲染成公众号可用的格式。

### read-gzh - 读公众号

读取公众号文章并总结要点。

### brush - 图片生成

生成文章配图。
