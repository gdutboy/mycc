# Draft - 文章初稿生成

> 根据大纲写文章初稿。

## 触发词

- "写初稿"
- "起草"
- "draft"
- "帮我写文章"

## 输入

用户提供：
- 文章主题
- 大纲（可选，没有则先生成大纲）
- 风格要求（可选）

## 输出

初稿文章（Markdown 格式），包含：
- 标题
- 引言
- 正文（按大纲展开）
- 总结

## 初稿原则

### 1. 内容完整

- 按大纲展开每个点
- 不遗漏重要论点

### 2. 表达清晰

- 口语化表达
- 短句为主
- 避免复杂句式

### 3. 例子丰富

- 每个观点配例子
- 例子具体生动

### 4. 情感真实

- 有温度
- 有态度
- 不端着

## 输出格式

```markdown
# [标题]

## 引言
[引入主题，点明价值]

---

## [正文]

### xxx
[内容展开]

---

## 总结
[核心观点回顾]
```

## 后续流程

初稿完成后，可以：
1. 用 polish 精修
2. 用 gzh 排版
3. 用 title 改标题
