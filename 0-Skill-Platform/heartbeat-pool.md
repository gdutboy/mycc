> cc 心跳巡逻的扫描目标。每次心跳唤醒，从池中挑饥饿值最高的焦点执行。
>
> **饥饿值公式**：`hunger = wait_hours × frequency_weight`

## 频率参考表

| 频率关键词 | frequency_weight | 说明 |
|-----------|-----------------|------|
| 每小时 | 4.0 | 高频，紧急事项 |
| 每2小时 | 3.0 | 较高频 |
| 每日 | 1.0 | 常规日检 |
| 每3天 | 0.5 | 低频 |
| 每周 | 0.2 | 周期性回顾 |
| 每月 | 0.05 | 月度检查 |

## 管理规则

- **添加焦点**：主 cc 决定，ID 按 f{数字} 递增（f12 为 cc-todo 扫描，固定保留）
- **删除焦点**：一次性焦点完成后，巡逻员在回报中提醒，主 cc 删除
- **修改频率**：根据实际需要调整，同步更新频率参考表
