# wechat-decrypt 工具研究

> 来源：公众号文章（2026-03-10）
> 用途：开发参考 / 技术调研

## 核心能力

直接操作微信数据库层面，获取原始 SQLite 文件访问权限，实现：
- 全量聊天记录的全文搜索
- 用标准 SQLite 工具做程序化访问
- 官方工具没有的实时消息流
- 通过 MCP 接入 AI

## 技术要点

| 要点 | 说明 |
|------|------|
| 原理 | 读取微信进程内存，解密数据库 |
| 权限 | 需要管理员权限（所有平台都如此） |
| 依赖 | Python 环境、运行中的微信进程 |
| 限制 | 只能解密自己的微信数据 |

## 适用场景

- 个人数据恢复和备份
- 聊天记录全文检索
- AI 接入（ MCP ）做智能助手

## 与官方工具对比

| 维度 | 官方备份 | wechat-decrypt |
|------|----------|----------------|
| 数据格式 | 官方格式 | 原始 SQLite |
| 全文搜索 | 不支持 | 支持 |
| 程序化访问 | 有限 | 完全支持 |
| 实时消息 | 不支持 | 支持 |
| 操作复杂度 | 简单 | 较高 |

## 合规声明

> 仅用于解密自己的微信数据，遵守所在地法律法规。不是监控工具。
