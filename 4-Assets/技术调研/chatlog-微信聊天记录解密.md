## 核心功能

- **解密数据库**：获取微信密钥，解密聊天数据库
- **导出聊天记录**：支持 Excel/CSV 格式
- **HTTP API**：提供 REST API 查询
- **MCP 服务**：可与 AI 助手集成（ChatWise、Cherry Studio、Claude Desktop）
- **支持平台**：Windows + macOS
- **微信版本**：3.x / 4.x
