# 4-Assets 可复用资产

> 存放可以重复使用的资源

---

## 目录

| 目录 | 说明 |
|------|------|
| `模板/` | 项目模板、README模板 |
| `SOP/` | 标准操作流程 |
| `Prompt/` | AI Prompt 模板 |
| `设计文档/` | 系统设计文档 |
| `学习资源/` | 学习资料 |
| `技术调研/` | 技术调研报告 |

## 核心原则

**可复用** —— 放进来的东西要能多次使用，不然就是 Inbox。

## 使用方法

1. 启动新项目 → 复制 `模板/` 到项目目录
2. 写代码前 → 参考 `SOP/`
3. 让 AI 干活 → 使用 `Prompt/` 模板
